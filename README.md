## O que é o TextEditor?
A ideia principal do TextEditor é criar, salvar e abrir arquivos de texto por meio do código e terminal.

## Pré-requisitos
- .NET

## Como rodar o código?

### [1] Primeiro, rodar os comandos dotnet
```bash
dotnet restore
dotnet build
dotnet clean 
```

### [2] Segundo, rodar a aplicação
```bash
dotnet run
```
